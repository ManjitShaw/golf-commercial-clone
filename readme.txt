# 🎯 Sidcup Family Golf - Landing Page Clone

A visually appealing, animated landing page clone of the **Sidcup Family Golf** website. Built using modern front-end technologies and animation libraries.

## 🛠️ Tech Stack
- HTML
- CSS
- Vanilla JavaScript
- GSAP (GreenSock Animation Platform)
- GSAP ScrollTrigger Plugin

## 💡 Features
- ✨ Custom animated cursor (including blur effect)
- 🎞️ Smooth scroll-based animations with GSAP
- 🧭 Animated navigation and interactive hover effects
- 📜 Scrolling marquee-like banner section
- 🌫️ Parallax-style blurred background animation

## 🚀 Future Improvements
- 📱 Make it fully responsive for mobile & tablets
- 🧩 Add more interactive sections and elements
- ⚡ Optimize animation performance and load time
